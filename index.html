##字体
|字体种类|特点|
|:—–:|:–:|
|衬线字体|强调横竖笔画的对比，笔画起止处有修饰，有优雅和历史感；远处观看识别性低，同一字号看起来更小，视觉冲击力差|
|非衬线字体|无弱化效果，视觉冲击力更强|

正文使用非衬线字体！

###字体推荐：

####中文
1.严肃
正文使用微软雅黑
标题：黑体、方正粗宋、方正综艺简体、华丽俪金黑

2.轻松

  • 方正北魏楷书
  • 方正静蕾简体（手写体）
  • 康熙字典体（繁体使用）
  • 方正稚艺简体
  • 文鼎习字体（不适合大面积使用）

####英文

  • 微软雅黑
  • Times New Roman（有历史感）
  • Tahoma（圆润，亲切感）
  • Segoe UI Light（纤细/扁平化）

在一页中，字体少于三种；字数不要太多；使用合适的配图进行解释

##图片

###使用形式

  1. 照片
  2. 剪影
  3. 图标
  4. 简笔画
  5. 3D小人
  6. 2D小人
  7. 剪贴画

###处理技术

  1. 裁剪
  2. 调整大小
  3. 修正图片基本参数（图片工具-格式-调整-更正-图片更正选项）
    • 亮度
    • 对比度
    • 清晰度
    • 饱和度
    • 色温
  4. 重新着色（图片工具-格式-颜色-重新着色）
    • 冲蚀效果之后便于添加文字
    • 单一颜色可以使图片更纯粹，可使用多个单一颜色进行组合
    • 灰度可以将协调多个图片的色彩冲突，适合于商务
  5. 艺术效果
    • 虚化背景可以在突出文字的同时保留更多的颜色；也可以用作“画中画”，以突出某一图片元素
  6. 多种图片效果，如阴影，柔化边缘等
  7. 版式

###注意点

  • 尺寸不得过小
  • 不能随意摆放
  • 大图尽量覆盖全屏
  • 使用清晰的图片
  • 去除水印
  • 使用shift调节大小
  • 不使用无关图片
  • 在使用颜色相近的图文时，在文字下方加入色块

####n*n拼图
裁剪-纵横比-1:1

####设置透明色

  • 颜色背景单一时
  • 背景与抠出的图片有差异
  • 抠出的图片缩小后呈现

####图片安排

  1. 人物视线
    • 单个人物有文字时视线向字
    • 多个人物视线相对
  2. 风景图上天下地
  3. 图片内部内容也应对齐

####蒙版
文字与图片相互独立时，可在图片上加入一个由白色到无色的渐变色块

####3B原则（Beauty,Beast,Baby）

颜色相对时，文字突出，强调作用明显

##颜色

###配色知识

####色彩三要素

  1. 色相
  2. 亮度
  3. 饱和度
    ####三原色和色轮
    ####冷暖色
  • 冷色：青，蓝
  • 暖色：红，橙，黄
  • 中性：绿，紫

###配色方案一：色彩意义

####色彩意义

  1. 红

    积极，热情，健康，活泼

    愤怒，敌对，危险

  2. 橙

    健康，温暖，舒适

  3. 黄

    智慧，权力，骄傲，青春，警示

  4. 绿

    和平，环保，生命，安全，年轻，希望

  5. 青

    坚强，希望，古朴，庄重

  6. 蓝

    平静，理智，低调，冷淡

  7. 紫

    神秘，高贵，优雅，具备极强的女性气质。

  8. 白

    纯洁，干净，神圣

  9. 黑

    高贵，肃穆，华丽

  10. 灰

    正确，严谨，商务

    消极，平凡，中庸

####行业配色

  1. 高科技行业：黑、灰、蓝
  2. 金融财会：深蓝、深红
  3. 制造业：蓝、黑、白、灰
  4. 医药：绿、蓝
  5. 党政：深红、黄、深蓝
  6. 教育：黄、橙、绿

###配色方案二：专业配色方法

####零度对比（同一色相之间或与黑白灰的搭配）

  1. 无彩色

    只用黑白灰。对比效果大方、庄重、高雅、富有现代感，但也易产生单调感

  2. 无色彩与有色彩

    大方又活泼，无彩色面积大时高雅、庄重，有彩色面积大时活泼感加强。

  3. 同类色

    对比效果统一、文静、雅致、含蓄、稳重，但易于单调，呆板

  4. 无彩色与同类色对比

    综合了2和3
    ####调和对比

  5. 邻近色
    色相环上距离30°,弱对比类型。需要调节明度差
  6. 类似色
    距离60°,较弱对比
  7. 中度色
    距离90°

####强烈对比

  1. 对比色
    距离120°,需要多种调和手段
  2. 互补色
    距离180°

####多色相对比

  1. 单页面内容区分
  2. 多页间层次区分
